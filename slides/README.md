# Slides

ここには発表用の素案を置きます。

基本方針:

- 発表用の文面は英語で書く
- 元ログは日本語でよい
- まずは Markdown で素案を作る
- 図表や画像の候補はログの `参照・成果物` と H2 の `dataset_registry.md` から拾う
- 1 回の発表につき 1 ファイル

補足:

- 日々の研究メモは日本語で蓄積する
- スライド化するときに英語へ変換する
- 必要なら Codex が日本語ログから英語のスライド文面へ整える


自動生成の入口:

```powershell
python scripts/build_slide_outline.py --from 2026-04-21 --to 2026-04-27 --title "研究立ち上げ週の進捗"
```

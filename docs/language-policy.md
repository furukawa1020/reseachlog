# Language Policy

## Default Rule

- Research logs: Japanese
- Chat updates: Japanese
- Slide outlines and presentation text: English

## How to use this with Codex

あなたは日本語で進捗を書いてよい。
Codex はその内容を次の 2 つへ変換する。

1. `logs/daily/` の研究ログ
2. `slides/outlines/` の英語スライド素案

## No need to repeat every time

毎回
`ログは日本語、スライドは英語`
と言い直さなくてよい。

この repo ではそれを既定の運用ルールとする。

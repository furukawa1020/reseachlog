# Research Plan

## テーマ

生理反応と主観的意味づけの可変性を扱い、
`同じ生理反応がなぜ異なる意味に読まれるのか`
を、センシング・解析・外在化インターフェースを通して観察可能にする。

## 中核となる問い

- 生理反応と主観的意味づけの対応は一対一なのか
- どの条件でその対応は成立し、どの条件で破綻するのか
- 外在化インターフェースはその差異を観察可能にできるのか

## 仮説

### H1

既存研究のメタレビューにより、
生理反応と主観的意味づけの対応は、
文脈や個人差に依存して可変的に成立・破綻する条件が整理できる。

### H2

WESAD を軸とした公開データ解析により、
同一の生理反応パターンに対して主観ラベルが一意に対応しない例を示せる。

補足:

- 研究計画上の主軸は WESAD
- リポジトリ運用上は CASE など他データセットも比較対象として拡張可能

### H3

外在化インターフェースは、
質問紙や言語報告だけでは捉えにくい意味づけの差異を観察可能にする。

## 2026 年の主戦場

研究計画 PDF から見る今年の中心は以下です。

1. H1 のメタレビュー
2. H2 の WESAD 論文
3. `n=20〜30` の追試設計
4. Lazarus 理論との接続とモデル化

この repo も、この 4 本を毎週前に進めるための形にする。

## 作業の柱

### H1 メタレビュー

- ストレス研究
- 生理指標研究
- 認知的評価と reappraisal
- 文脈依存性と個人差

必要な成果物:

- 論文サーベイ表
- 条件整理の図
- 使う概念の定義メモ

### H2 データ解析

- WESAD のデータ入手と前処理
- ラベル定義と生理特徴量の確認
- `ズレ` の定義
- 破綻例の可視化
- CASE などへの横展開余地の確認

必要な成果物:

- WESAD 解析手順
- データセット比較表
- 解析ノート
- 可視化図
- 暫定結果メモ

### H3 外在化インターフェース

- 既存プロトタイプの整理
- 最小構成の決定
- 比較観察実験の設計
- 意味づけ差異の記録方法の確定

必要な成果物:

- 試作メモ
- 実験プロトコル
- 観察ログ
- 図版候補

## 資金・社会実装の整理

新しい研究計画では、資金面は次のように位置づけられている。

- 大学内学生研究助成
- 民間研究費への応募
- 研究プログラム等への参加
- 未踏等は `技術発展の可能性として位置づけ`

つまり、当面の中心は `研究の芯を作ること` で、
資金獲得や社会実装はその延長として段階的に進める。

## ロードマップ

### 直近

- H1 のレビュー軸を決める
- H2 の WESAD 解析パイプラインをつくる
- Lazarus 理論との接続点を明文化する
- H3 の最小試作と観察計画を固める

### 中期

- H1 / H2 の論文化の芯をつくる
- H3 の小規模実験で定性・定量データを取る
- `n=20〜30` 追試の準備をする

### 長期

- 意味づけを扱う測定・設計フレームとして体系化する
- HCI / 認知科学 / 倫理・哲学まで横断して展開する
